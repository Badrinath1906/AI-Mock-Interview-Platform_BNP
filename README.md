# 📌 Overview

AI Mock Interview Platform is an AI-powered web application that simulates real technical and HR interviews.

The platform generates company-specific interview questions using **Google Gemini AI**, allows users to answer questions, provides AI-generated sample answers, evaluates responses, generates a detailed performance report, and stores interview history in a MySQL database.

It is designed to help students and job seekers improve their interview skills and prepare for placements with a realistic interview experience.

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/Badrinath1906/AI-Mock-Interview-Platform_BNP.git
```

## Move into Project Folder

```bash
cd AI-Mock-Interview-Platform_BNP
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run Application

```bash
python app.py
```

or

```bash
flask run
```

---

# 🔑 Environment Variables

Create a `.env` file in the project root.

```env
GOOGLE_API_KEY=YOUR_GEMINI_API_KEY

MYSQL_HOST=localhost
MYSQL_USER=root
MYSQL_PASSWORD=your_password
MYSQL_DATABASE=interview_db

SECRET_KEY=your_secret_key
```
